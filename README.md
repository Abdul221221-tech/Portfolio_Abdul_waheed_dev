# Portfolio Website

This project is a modern personal portfolio website built with **Next.js** and **React**. It showcases my skills, projects, experience, and contact information in a visually engaging way using animations and interactive sections.

## What this project does

- Displays a professional hero section and personal introduction
- Highlights technical skills and areas of expertise
- Shows featured projects and work experience
- Includes a contact section with Firebase-backed form handling
- Uses animations and 3D/interactive visuals to make the site more engaging

## How to run the project

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and visit:
   ```text
   http://localhost:3000
   ```

## Useful commands

- `npm run dev` - run the app locally
- `npm run build` - create a production build
- `npm run start` - run the production build
